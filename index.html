Threads

  • Smallest unit of execution to which processor can allocate time
  • Threads live within process
  • Threads can interact with shared resources
  • Able to read/write to different memory addresses

Type of Threads

  1. User Level Threads : 
Threads controlled by user, user can create, use , kill these threads
  1. Kernel Level Threads :
very low level threads controlled by operating system

Multithreading

Multithreaded processor can run multiple threads on a single core at the same time (At least this is what we see)
How Multithreaded processor achieve this on a single core ?
The answer is context switching
